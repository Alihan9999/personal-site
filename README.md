# Alihan Cakiralioglu Personal Site

Production-style portfolio site for a DevOps / Platform Engineer built with React, Vite, Tailwind CSS, and Framer Motion.

## Local Development

1. Install dependencies:

```bash
npm install
```

2. Start the dev server:

```bash
npm run dev
```

3. Open the local URL shown by Vite.

## Build

```bash
npm run build
```

## Deploy To Vercel

1. Push this repository to GitHub.
2. Import the repository into Vercel.
3. Keep the default settings:
   - Framework preset: `Vite`
   - Build command: `npm run build`
   - Output directory: `dist`
4. Deploy.

## Notes

- Main app entry: `src/App.jsx`
- Reusable UI components live in `src/components`
- Content data for projects, blog posts, skills, and contact links lives in `src/data.js`
